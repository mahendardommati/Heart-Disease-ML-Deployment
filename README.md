# Heart Disease Prediction using Machine Learning

This is an end-to-end Flask web application that predicts heart disease using multiple ML models like:
- KNN
- Logistic Regression
- Naive Bayes
- Decision Tree
- Random Forest
- AdaBoost
- Gradient Boosting
- XGBoost
- SVM

## Tech Stack
- Python, Flask
- HTML, CSS
- scikit-learn, XGBoost, NumPy, Pandas

## How to Run
1. Clone the repository
2. Install dependencies
3. Run `python app.py`
4. Open `http://127.0.0.1:5000/`
